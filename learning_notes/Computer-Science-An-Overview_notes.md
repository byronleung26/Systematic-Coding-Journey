# 第1章 数据存储
1. 位和位存储
- 位（bit，binary-digit，二进制数字）
- 布尔运算（Boolean operation）
- 门（gate），也称逻辑门（logic gate）
- 触发器（flip-flop），计算机存储的基本单元
- 超大规模集成（very large-scale integration，VLSI）——>芯片（chip）
- 长位串称为流（stream），十六进制计数法（hexadecimal notation）简写表示
2. 主存储器（main memory）
- 存储单元（cell），字节（byte）
- 最高有效位（most significant bit），高位端（high-order end），低位端（low-order end），最低有效位（least significant bit）
- 地址（address）
- 读操作，写操作
- 随机存取存储器（random access memory，RAM），动态存储器（dynamic memory，DRAM），同步DRAM（synchronous DRAM，SDRAM）
- 1kilobyte（KB）=1024byte，megabyte（MB），gigabyte（GB），terabyte（TB）
3. 海量存储器（mass storage）
- 磁盘（magnetic disk），硬盘驱动器（hard disk drive，HDD）
- 光盘（compart disk，CD）
- 闪存驱动器（flash drive）
4. 用位模式表示信息
- 文本的表示，ASCII，UTF-8
- 文本文件（text file），文本编辑器（text editor），字处理程序（word processor）
- 数值的表示，二进制计数法（binary notation），二进制补码（two’s complement）记数法，浮点记数法（floating-point notation）
- 图像的表示  
①将图像解释为一组点，每个点称为一个像素（pixel，picture element简写），对每个像素的显示进行编码，整个图像就表示成了这些编码像素的集合，称为位图（bit map），RGB编码，使用一个“亮度”成分和两个颜色成分  
②将图像描述成了几何结构（如直线和曲线）的集合，这些几何结构可以用解析几何技术来编码
- 声音的表示  
①按有规律的时间间隔对声波的振幅采样，并记录所得到的值序列  
②乐器数字化接口（musical instrument digital interface，MIDI），对什么乐器演奏什么音符以及持续时间进行编码

# 模块1 C++语言简介
## C++语言发展史
1. C++语言是由Bjarne Stroustrup于1979年在美国贝尔实验室开始设计开发的，C with classes
2. 1983年更名C++
2. 1998年,C++的ANSI/ISO标准被投入使用,这是C++标准第一版,称为C++98
3. C++03、C++11、C++14、C++17、C++20
## C++语言的特点
1. 编译式的、通用的、大小写敏感的，完全支持面向对象的开发模式
2. 和C语言的异同点：  
C++是C语言的继承，尽量兼容C语言  
C++语言比C语言更安全  
C++语言的应用领域更加广泛  
C++语言可运行于多种平台上  
C++语言中加入了面向对象，C++的程序结构与C语言的程序结构存在很大差别
3. 基本的输入/输出
- cin  
istream类的对象，实现基本的键盘输入  
使用流提取运算符`>>`从键盘取得数据(提取操作)  
`cin>>变量1>>变量2>>...>>变量n`  
- cout  
ostream类的对象，实现基本的屏幕输出  
使用流运算符`<<`向输出设备屏幕输出信息(插入操作)  
`cout<<表达式1<<表达式2<<...<<表达式n`  
- getchar()函数
4. 头文件和命名空间
- `#include<iostream>`，标准输入输出流
- `#include<fstream>`，标准文件流
- `#include<string>`，标准字符串处理函数
- `#include<cmath>`，标准数学函数
- `#include "e:\myprog\ex1.h"`，自定义头文件
- 命名空间namespace，也称名字空间
    ```cpp
    namespace 命名空间名
    {
        命名空间内的各种声明(函数声明、类声明、.....)
    }
    ```
- std
- `命名空间名::标识符名`  
`using 命名空间名::标识符名`  
`using namespace 命名空间名`
5. 强制类型转换运算符
- static_cast，将一种数据类型转变为另一种数据类型  
`static_cast <类型名>(表达式)`  
`oneInt2 = static_cast<int>(oneDouble);`，推荐写法  
`oneInt2 = int(oneDouble);`  
`onelnt2 = (int)oneDouble;`  
`oneInt2 = oneDouble;`  
- const_cast，去除指针和引用的常量性  
`const_cast <类型名>(表达式)`  
将常量指针转化成非常量的指针，并且仍然指向原来的对象；或是将常量引用转换成非常量的引用，并且仍然指向原来的对象
6. 函数参数的默认值
- 指定默认值时不仅可以用常数，还可以用任何有定义的表达式作为参数的默认值。但不允许是函数内定义的局部变量  
- 提供默认值时必须按从右至左的顺序提供，即有默认值的形参必须在形参列表的最后  
- 在函数原型中,可以省略形参的名字，而只给出它的类型及默认值
- 函数参数的默认值可以写在声明函数的地方，也可以写在定义函数的地方，但不能在两个地方都写
7. 引用和函数参数的传递
- 引用的定义  
引用相当于给变量起个别名  
`类型名 &变量名 = 同类型的某变量名`  
不能有空引用，即引用必须指向某个已存在的内存区域的首地址。也不能声明引用的引用  
定义引用时需要进行初始化，通常会用它所“引用”的变量去初始化  
定义引用时，可以在定义的前面加const关键字，表明该引用是“常引用”  
`int oneInt`  
`const int &cname = oneInt`

## C++语言的结构
1. 以.cpp作为文件名，文件中包含若干个类和若干个函数
2. 必须有且仅有一个主函数main()，这是程序执行的总入口
3. 程序从主函数的开始处执行，按照其控制结构，一直执行到结束
4. 程序的结束通常是：1.在主函数中语法哦return语句，2.执行到主函数最后的括号}
5. 主函数中可以调用程序中定义的其他函数,但其他函数不能调用主函数。主函数仅是系统为执行程序时所调用的
6. C++程序中,仍沿用C语言的注释风格,即注释有以下两种形式：1.从/\*开始,到*/结束,这之间的所有内容都视作注释。2.从//直到行尾,都是注释。

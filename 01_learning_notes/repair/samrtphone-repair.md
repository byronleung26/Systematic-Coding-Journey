# 第1章 智能手机的结构与工作原理
## 1.1 智能手机的整体结构
### 1.1.1 智能手机的外部结构
- 显示屏  
- 听筒  
- 前置摄像头  
- 后置摄像头  
- 音量增/减键  
- 开关机键  
- 扬声器  
- USB/充电/耳机接口  
- 话筒  
- SIM卡槽  
### 1.1.2 智能手机的内部结构
1. 显示屏组件  
- 保护玻璃层  
- 触摸板层  
- 显示层  
- 背光层  
2. 电路板  
单层电路板/双层电路板  
主电路板/副电路板  
- 处理器芯片  
中央处理器，控制中枢  
双处理器，AP（应用处理器）和BP（基带处理器）  
单处理器  
- 射频芯片  
负责信号的收/发、调制和解调  
- 射频功率放大器芯片  
对于射频信号进行功率放大，耗电量较大  
- 电源管理芯片  
对电池供电电压进行变换/分配/检测/输出及电池充电管理控制等  
- 存储芯片  
Flash闪存，RAM随机存储器，ROM只读存储器  
3. 常见功能部件  
以独立形态存在于智能手机中，通过接口或触点压接的形式与电路板连接，形成供电和控制关系  
- 摄像头  
- 扬声器  
- 振动器  
- 听筒/话筒  
- 传感器  
- 屏幕指纹识别模块  
- 听筒  
- 按键  
- 插孔与接口  
4. 电池

1. 智能手机的整机控制过程  
- 手机信号接收的控制过程  
- 手机信号发送的控制过程  
- 手机其他功能的控制过程  
2. 智能手机的电路关系  
- 智能手机的整机控制关系  
- 射频电路、语音电路、微处理器及数据信号处理电路之间的关系  
- 接口电路与微处理器及数据信号处理电路之间的关系  
- 电源电路与各单元电路的关系  
- 其他功能电路与各电路之间的关系

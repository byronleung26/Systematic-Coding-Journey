## 4.2 模块和包
- 模块
- 包，包含一个或多个模块的目录。该目录必须包含一个名为__init__.py的文件（可以为空）
- 模块实际上是对象，而不仅仅是文件
- 如果你在包中未创建__init__.py文件，它将变成一个隐式的命名空间包。命名空间包允许将包分发为多个部分

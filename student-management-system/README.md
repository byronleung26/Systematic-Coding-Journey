# 项目概述

1. 项目名称  
   学生信息管理系统（student-information-management-system）。
2. 开发环境
 - 操作系统：WSL（Ubuntu）
 - 开发工具：VS Code
 - 语言版本：Python 3.13.11
 - 版本控制：Git
3. 项目定位  
   基于Python语言开发的轻量级学生信息管理系统，实现学生信息的录入、维护、查询、排序及数据持久化功能，适用于学习Python基础语法、模块化编程及文件操作的实战项目。

# 需求分析

1. 添加学生及成绩信息
2. 将学生信息保存到文件中
3. 修改和删除学生信息
4. 查询学生信息
5. 根据学生成绩进行排序
6. 统计学生的总分

# v1.0版本说明
1. 核心目标：实现学生信息的基础管理功能
2. 主要功能：
 - 学生信息录入
 - 删除学生信息
 - 修改学生信息
 - 按学生姓名、ID查找
 - 查询并显示学生信息
 - 统计学生人数
 - 根据各科成绩和总成绩进行降序和升序排列
3. 技术实现
 - 采用 Python 基础语法实现核心逻辑
 - 使用字典存储单条学生信息，列表管理所有学生数据
 - 通过文件操作实现数据持久化（保存 / 加载）
 - 模块化设计，将功能拆分为独立函数，便于维护与扩展
 - 提供命令行交互界面，操作简单直观
